## Описание
###### Description
---
Трекер дел - программа, объединяющая возможности приложений для ведения списков дел, календарей и трекеров задач. Позволяет легко и быстро организовать свое время, не беспокоясь о возможности забыть выполнить что-нибудь необходимое. С помощью данного приложения пользователю не придется запоминать все множество рутинных дел, а предоставить эту задачу программе, которая в удобной форме организует и оптимизирует ваш распорядок. Программа позволит избавиться от множества календарей и планировщиков, тем самым собрав все необходимые функции в одном приложении.
Программа будет написана на языке программирования Python. 


## Аналоги
###### Analogues

Рассмотрим наиболее известные аналоги приложения.  
**Программы, позволяющие организовать списки дел**:  
1) Wunderlist  
_Version 3.4.8 build #1142_. Стандартная версия для Android.  
Первое приложение из данного вида программ, которое я начал использовать. Для начала необходимо завести аккаунт, после этого можно работать с приложением. Рассмотрим его возможности.

Приложение позволяет выбрать категорию списков дел: Работа, Семья, Просмотр кино, дела, которые необходимо выполнить сегодня, и другие, либо задать свое название категории. При необходимости можно пригласить других пользователей для совеместного выполнения дел. Есть возможность создавать папки, куда помещаются различные категории списков дел. Списки дел можно дублировать. Доступ к спискам дел можно получить с любого устройства. Есть система оповещений, уведомляющая о добавлении, создании файлов людьми с которыми вы сотрудничаете. Важные дела можно отмечать как избранные, добавлять фотографии, презентации и другие документы. Реализован быстрый поиск дел по всем категориям. Можно установить срок выполнения дела. При необходимости есть возможность перенести дело из одной категории в другую. Можно начать беседу, комментируя дела людей, с которыми вы сотрудничаете. Для того, чтобы  оповестить систему о завершении дела необходимо просто поставить галочку рядом с названием дела, после чего дело автоматически удаляется из списка. На любой вкус можно выбрать фон для приложения. Можно выполнить интеграцию с социальными сетями Facebook и Google+. Существует платная Pro версия приложения предоставляющая неограниченный доступ к файлам, описывать предназначение и создавать подзадачи для более успешного выполнения задачи.

На основе обзора приложения можно выделить следующие положительные стороны и недостатки:  
__Плюсы__:  
1. Простой и приятный интерфейс.  
2. Возможность объединять дела в папки.  
3. Приглашение пользователей для совместного выполнения дел.  
4. Доступ с любого устройства.  
__Минусы__:  
1. Если по ошибке отметить дело как выполненное, нет возможности его восстановить.
Для себя я выделил следующие __полезные возможности__:  
1. Объедение категорий списков дел в папки.  
2. Совместное выполнение дел с другими участниками.  
3. Поиск по делам.    
4. Добавление фотографий и другого мультимедиа для дел.  
5. Установка срока выполнения дела.  

2) Todoist  
_Version 12.6_. Стандартная версия для Android.  
Для использования необходимо наличие аккаунта. Рассмотрим возможности.

Существует статистика пользователя, отражающая общее количество выполненных дел, количество выполнений задач в каждый определенный день и уровень пользователя в соответствии с успехами в выполнении дел: Начинающий, Новичок, Профессионал, Эксперт и так далее. К сожалению множество других функций статистики, такие как просмотр детаей выполнененых задач, требует наличие премиум версии приложения. Так же премиум версия открывает следующие возможности: комментарии, улучшенная система фильтров, синхронизация с календарем, напоминания, лейблы для дел, запись аудио, местоположения, возможность возврата выполненных дел и многое другое. Стандартная версия включает в себя функции добавления дела в список, при этом каждое дело можно фильтровать по категории и приоритету, определить время к которому необходимо выполнить поставленную задачу. Есть в приложении и поиск дел. Посмотреть список дел можно с учетом фильтров и приоритетов. Есть хранилище выполненных дел. По желанию можно сменить тему приложения. Есть опция карма, которая позволяет задать количество дел, которые необходимо выполнить в день.

На основе обзора я выделил следующие __положительные стороны__:  
1. Статистика пользователя.  
2. Система фильтров.  
3. Опция карма.  
4. Наличие пользовательского гайда по приложению.
__Минусы__ приложения:  
1. Множество полезных функций требует премиум версии.  
2. По сравнению с предыдущем аналогом, более сложный и не привлекательный интерфейс.  
Для себя я выделил следующие __полезные возможности__:  
1. Статистика.  
2. Установка приоритета для дела.  
3. Опция карма.  

3) Any.do  
Приложение требует наличия аккаунта. Рассмотрим возможности и особенности программы.  
При первом запуске приложения появляются интерактивные подсказки, которые позволяют быстрее освоить функции приложения. При добавлении можно воспользоваться функцией голосового ввода.После добавления задачи ее можно редактировать, при этом появляются функции добавления подзадач, перемещения в другую категорию подзадач, установка оповещения, прикрепления к задачи описания, фотографий, документов, видео, аудио и другого, есть возможность присоединить подключить к выполнению задачи другого пользователя. После отмеки о выполнении задачи, она не исчезает полностью, для того чтобы ее убрать необходимо ее удалить. Можно подключиться к календарю и импортировать оттуда список задач, которые необходимо выполнить в определенный день. Задачи можно сортировать по времени и по списку. Есть замечательная функция "Спланировать день", которая предоставляет возможность создавать задачи на текущий день и по мере выполнения производить уточнения о сроках выполнения и все это выполнено в дружественном пользователю интерфейсе.
Есть премиум-версия. Её возможности: теги и управление цветами, напоминания привязанные к местонахождению, неограниченное по приглашениям совместное списки, повторяющиеся задачи, возможность персонализировать приложение, VIP-поддержка.

__Плюсы__ приложения:  
1. Подсказки для новых пользователей.  
2. Голосовой ввод.  
3. Удобная система редактирования задачи.  
4. Создание подзадач.  
5. Импорт задач из календаря.  
6. Приятный дружественный пользователю интерфейс.  
__Минусы__ приложения:  
1. Много полезных функций доступно только в премиум-версии.  
Для себя выделил следующие __полезные возможности__:
1. Создание подзадач  
2. Подсказки для новых пользователей  
3. Редактирование задач с использованием фильтров, приоритета и другое  

**Программы-календари**:  
1) Google calendar  
Очень удобный календарь. Позволяет добавлять цели, напоминания и события. При добавлении цели можно выбрать тип цели, например Пробежка, периодичность выполнения цели, длительность и время суток, после этого программа автоматически подберет для вас наиболее удобное время, которое в любое время можно отредактировать. Можно добавить напоминание, установив время, когда это напоминание необходимо. При добавлении события, можно указать время, на протяжении которого будет происходить событие, место проведения, при необходимость есть возможнось пригласить других людей, а так же прикрепить необходимый документ. Сам календарь представляет собой таблицу, для которой можно выбрать режим отображения дней: выводить по 1, 3, по 7 или весь месяц. Таблица показывает события, которые будут проходить в этот день. По мере выполнения событий их помечать, как выполненные. Существует возможность импорта событий, напрмер расписания занятий в университете или дни рождения родственников. Есть временная черта, показывающая текущей положение времени по временной шкале суток

__Плюсы__ календаря:  
1. Приятный интерфейс  
2. Возможность добавления событий, напоминаний и целей с множеством характеристик.  
3. Импорт расписаний и событий из различных источников.  
4. Поиск по событиям.  
__Минусов__ каледнаря не обнаружил.  
__Полезные возможности__, которые можно использовать в своем проекте:  
1. Импорт расписаний  
2. Поиск по событиям  
3. Отображение календаря таблицей по 1, 3, 7, 30/31 день со всеми событиями.  
 
2) Yandex calendar  
Позволяет добавлять события, указывая для них описание, время и дату, место проведения, добавлять участников, учитывая занятость каждого из них, создавать уведомления о событии, статус и доступ к редактированию события. Есть возможность создать новый календарь для отдельных событий.

__Плюсы__:  
1. Простой интерфейс  
__Минусы__:  
1. По сравнению с Google calendar имеет гораздо меньше возможностей  
2. Нет приложения для Android  
Среди __полезных возможностей__ выделил проверку занятости пользователей, приглашенных на событие.
